Product Name

The Bookbag

Application Overview

The Bookbag is an application that allows the user to track their book current, past and present. It allows the user to rank them by date they read, how many stars etc. The user is able to put their books inot categories of currently reading, 1 star, 2 stars, 3 stars, 4 stars, 5 stars, 6 stars and DNF. Among other features.

Technology Overview

Languages that will be used Python, CSS, JavaScript, HTML. The platform will be through Github at "https://yourusername.github.io/Bookbag".

Current and Future Feature List

Current features include: keeping track of book and reviews of books, sharing thoughts with others, organization, profile modification. Future features to look out for include: Able to have friends throughout the app and being able to create book club groups within the app for those who don't have time to meet in person.
